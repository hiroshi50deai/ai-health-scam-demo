# AI健康詐欺LP 解剖デモ

AI時代に、健康詐欺LPがどれほど医療っぽく・専門家っぽく・家族を不安にさせる形で作れてしまうかを伝える、注意喚起用の静的デモサイトです。

## 重要な注意書き

これは注意喚起のための架空デモサイトです。商品販売・医療提供・個人情報収集は行っていません。

## コンセプト

本物の販売ページではなく、AIが作り得る健康詐欺LPの構造を、監査レポート・解剖図・赤ペン添削として見せるサイトです。

- ファーストビューで「医療っぽさがAIで量産される」危険を提示
- 架空のブラウザ画面カードで危険な広告構造を分析
- LPの流れ、根拠の階段、赤ペン監査を通じて冷静に読み解く
- 親や家族に送れるクリック式チェックリストとX投稿文を用意

## 実装ファイル

- `index.html`: サイト本体
- `style.css`: 監査レポート風のレスポンシブデザイン
- `script.js`: スクロール、チェックリスト、X投稿文コピー機能
- `README.md`: この説明ファイル

## 安全設計

このデモサイトでは、以下を行いません。

- 商品販売
- 医療提供
- 個人情報収集
- フォーム設置
- メール入力欄の設置
- 決済への誘導
- 実在する医師、医療機関、研究機関、商品名の表示
- 効果を断定する商品説明
